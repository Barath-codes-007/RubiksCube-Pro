# RubiksCube-Pro
A fully interactive 3D Rubik's Cube simulator built with JavaScript, HTML, CSS, and Three.js. Features multiple cube sizes (2×2–5×5), smooth layer animations, timer, move counter, TPS tracking, light/dark mode, sound effects, shuffle, and auto-solve functionality.
